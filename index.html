Linea De Tiempo Del Relato
- Explicar qué es la comprensión de textos narrativos y su importancia
- Presentar las técnicas para la comprensión lectora (identificación de ideas principales, inferencias, etc.).
Ficha De Proyecto Transmedia
- Seleccionar un texto narrativo adecuado para el nivel de los estudiantes.
- Leer el texto en clase, realizando pausas para discutir el contenido utilizando el método del diálogo socrático.
Ficha De Intertextos
- Dividir a los estudiantes en grupos pequeños.
- Asignar tareas específicas de análisis de texto
- Utilizar recursos digitales para realizar actividades

Creativo

- Subproceso: Estrategias didácticas para la enseñanza del castellano.
- Contenido temático: Métodos de análisis de texto en grupo.
- Objetivo: Desarrollar la capacidad de los estudiantes para comprender y analizar textos narrativos,
utilizando técnicas efectivas y estrategias didácticas adecuadas.
